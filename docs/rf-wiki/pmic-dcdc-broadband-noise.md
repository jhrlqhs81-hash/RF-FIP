PMIC or DC-DC noise coupling is suspected when the receiver noise floor rises broadly with load, charging, display brightness, CPU activity, or regulator mode changes. Compare function ON/OFF, load sweep, regulator frequency, conducted and OTA baselines, and shielding or decoupling A/B.

Broadband noise should be distinguished from narrow spurs. The next analysis step should include noise floor comparison and isolation of the active power source.
